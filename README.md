# Оценка эффекта коммуникации «Набегай на абонемент»

## 1. Задача

Оценить, изменилась ли суммарная дистанция клиентов на беговой дорожке (`kms`) после отправки коммуникации участникам группы B.

## 2. Данные

- `group_a.csv` — список user_id контрольной группы (без коммуникации)
- `group_b.csv` — список user_id тестовой группы (получили коммуникацию)
- `data.csv` — дистанция `kms` для всех клиентов

## 3. Подготовка данных

- Данные объединены по `user_id`
- Удалены пересекающиеся клиенты (попали в обе группы) — 44 человека
- Удалены дубликаты внутри групп
- Выбросов по правилу 1.5×IQR не обнаружено
- Распределение `kms` бимодальное (пики в 0-10 км и 50-70 км)

**Итоговый размер групп:**
- Группа A: 1955 клиентов
- Группа B: 1956 клиентов

## 4. T-тест

```python
from scipy.stats import ttest_ind

result = ttest_ind(group_A.kms, group_B.kms)
Результат:

t-статистика: -0.28

p-value: 0.78
```
Вывод: p-value > 0.05 → нулевая гипотеза о равенстве средних не отвергается. Статистически значимого эффекта нет.

5. Бутстрап (собственная реализация)
Итераций: 10 000

На каждой итерации: выборка с возвращением размером с исходную группу

Расчёт разности средних (B — A)

Результат:

Медиана разности: 0.26 км

95% доверительный интервал содержит 0

Вывод: бутстрап подтверждает отсутствие значимого эффекта.

6. Минимальный детектируемый эффект (MDE)
Параметры:

α = 0.05

Мощность (power) = 0.80

n_A ≈ 1955, n_B ≈ 1956

Через statsmodels:
```python
from statsmodels.stats.power import tt_ind_solve_power

effect_size = tt_ind_solve_power(
    nobs1 = 1955,
    ratio = 1956/1955,
    alpha = 0.05,
    power = 0.80
)
```
Cohen's d = 0.09

По формуле (pooled_std):

Объединённое стандартное отклонение = 29.12 км

MDE (км) = 0.09 × 29.12 ≈ 2.6 км

7. Общий вывод
T-тест и бутстрап показали: статистически значимого эффекта от коммуникации нет

Реальная разница средних: 0.3 км

Минимальный эффект, который мог обнаружить тест: 2.6 км

Даже если акция работала, текущий дизайн эксперимента не позволял заметить эффект меньше 2.6 км.

8. Рекомендация
Для обнаружения малых эффектов необходимо увеличить размер выборки или выбрать более чувствительную метрику (например, долю активных клиентов).
